Overview
========

``ftw.portlet.clock`` provides the sbb flash clock from sbb.ch


Install
======

- Add ``ftw.portlet.clock`` to your buildout configuration

::

  [instance]
  eggs =
    ftw.portlet.clock

- Run buildout

- Install ``ftw.portlet.clock`` in portal_setup
